# Leptospirosis Detector 

The Leptospirosis Detector App, developed using Flutter, is an innovative mobile application designed to assist in the early detection and management of leptospirosis. Users can input their symptoms, and the app uses a sophisticated algorithm to assess the likelihood of leptospirosis, providing immediate feedback and recommendations.
